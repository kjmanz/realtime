# ことばの森

同じ場所にいる3〜20人が、それぞれのスマホで遊ぶリアルタイム会話ゲームです。ほとんどの人には同じお題、参加人数に応じた少数の人には少し違うお題が届きます。質問は3問から始まり、必要なら1問ずつ最大10問まで追加できます。自由に会話し、最後に違うお題の人を予想します。

## 起動

Node.js 20以上で、外部パッケージのインストールなしに動きます。

```powershell
npm start
```

パソコンでは `http://localhost:3000` を開きます。同じWi-Fiに接続したスマホから遊ぶ場合は、パソコンのローカルIPアドレスを調べ、スマホで `http://パソコンのIPアドレス:3000` を開いてください。

## 遊び方

1. 進行役が部屋を作り、4桁のコードを伝えます。
2. 参加者はニックネームとコードで参加します。
3. 進行役がラウンドを開始し、全員が自分のお題を確認します。
4. 専用質問を使って自由に会話します。
5. 最後に少数派の人数が発表され、その人数分の参加者を予想します。
6. 投票分布とお題を確認したら、そのまま次のラウンドへ進めます。

部屋とラウンドの状態は現在メモリ上に保存されます。サーバーを再起動すると部屋は終了します。
